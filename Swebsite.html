Site

De website was direct iets anders niet alleen omdat het een andere taal was ook omdat je een heel andere aanpak moest gebruiken. Je moet gestuctureerd te werk gaan met je html die je schrijft en dan achteraf de css die je gebruikt. Alles moet heel precies zijn je kan niet zomaar alles een naam geven je moet soms met klasses werken en dan met een ID en niet verwisselen.


Mening

Ik vond het leuker als javascript omdat je echt ziet wat je doet en je snel kan verbeteren wat fout is. En je dit ook echt vaak ziet in de realiteit zo een website dus je kan je goed vergelijken met andere wesites. Ik denk dat de resultaten ook beter waren dus dat ligt in één lijn.
