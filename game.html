Game (deerhunt)


Dit is mijn game genaamd omdat er dus een hert is die je moet raken met sneeuwballen. Die sneeuwballen vertrekken van de hand en gaan richting jouw rood vizier en hopelijk raak je dan het hert. dan zou er dus een nieuw hert spawnen en kan je zo hogere scores halen.

Het project


Ik vond het een zeer moeilijk project want javascript is niet direct mijn ding. je moet er veel inbeelding voor hebben en een stukje kunnen designen voor jouw tekeningen. Daarom werkte de gam niet helemaal maar de code was wel correct.
